Superhero Colosseum


This is a library of different superheros that allows users to find information on them and shows their power stats such as strength, intelligence, durability, etc.

Features:

  • A dashboard that lists superheroes from different universes
  • Cards that displays all the heroes the api contains
  • Displays hero's statistics and information, and compare their abilities

Stacks:

HTML, CSS, Bootstrap, Javascript, ChartJS

Atmosphere


Atmosphere is a web application that uses your local weather forecast to generate and play a unique Spotify playlist.

Features:

  • Playlist generation based on weather
  • User-controlled music player including play, pause, and skip
  • Automatic location settings

Stacks:

HTML, CSS, Bootstrap, Javascript, React, NodeJS, GSAP, PUG

Created as a final project for a Web Development class.
